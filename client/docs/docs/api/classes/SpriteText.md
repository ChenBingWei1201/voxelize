A sprite that can be used to display text. This is highly inspired by the
[THREE.SpriteText](https://github.com/vasturiano/three-spritetext) library.

![Sprite text](/img/sprite-text.png)

## Hierarchy

- `Sprite`

  ↳ **`SpriteText`**
