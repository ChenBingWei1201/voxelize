# voxelize [WIP]

A well-optimized web-based voxel engine.

## Development

```bash
# clone the repository
git clone https://github.com/shaoruu/voxelize.git
cd voxelize

# download dependencies
yarn

# start development
yarn run dev

# start both frontend/backend demo
yarn run demo

```

visit http://localhost:3000
